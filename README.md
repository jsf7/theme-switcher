# theme-switcher
JavaScript theme switcher class


## Installation

1. Include it in your project

   ```js
   import { ThemeSwitcher } from "theme-switcher.js" 
   ```

## Usage

   ```js
   const themeSwitcher = new ThemeSwitcher()
   ```
